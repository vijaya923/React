Type Inference : Type inference in TypeScript is the ability of the compiler to automatically determine the 
datatype of a variable,function return value, object, or array based on its assigned value and usage, without requiring explicit type annotations.

Type compatability in TypeScript refers to the rules that govern whether one type can be assigned to another,
or used interchangeably in contexts like function arguments or return values.

Enums: Enums(enumerations) in TypeScript provide a way to define a set of named constants.They enhance code
 readability and maintainability by allowing the use of descriptive names instead of raw numeric or string values. 

 syntax:

   enumenumName {
    value1,
    value2,
    value3,


    ...
    valueN
   }
