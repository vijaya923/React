Rest Operator: It is used to store rest of the values passed to the function.

    Syntax:

      function functionName(param1, param2, ...restValues) {

      }

      functionName(value1, value2, value3)

      functionName(value1, value2, value3,Value4)

      functionName(value1, value2)

      functionName(value1, value2, value3, Value4, Value5)

      Note: restValues parameter should be last parameter in function.


Default Parameters: Used to assign default value to parameter i the function.

   Syntax:

       function functionName(param1 = defaultValue, param2 = defaultValue){

       }

       functionName();
