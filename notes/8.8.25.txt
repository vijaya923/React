Typescript: It is a programming language which is built on javascript.

programming language : whole instructions of a file will be executed at a time and returns the output.

Static Typing Indicates defining the data type while declaring the variable.

>It is a high level (Human understandable) programming language 

>It is built to develop large scale applications and transpites the code to Javascript.

  Transplation: converting the instructions from one programming language to another programming language.

> Browser won't support TypeScript, that's why we  need to convert the instructions from typescript to JavaScript to view the output

> It is developed by microsoft and released as open-source software.

> Typescript may be used by javascript to develop client side and server side applications.

  client side applications - React Js/ Angular / Vue Js

  server side applications - Node Js & Express Js

> Typescript compiler (tsc) will help us in transpiling instructions from Typescript to JavaScript.

> Current stable version is 5.8.2

> Initial release was on 1 october 2012

> File extension .ts (Typescript) / .tsx (TypeScript + XML)
